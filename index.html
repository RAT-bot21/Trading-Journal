Your journal saves automatically in your browser. To back up to iCloud, export your data as a JSON file and move it into your iCloud Drive folder. On iPhone and Mac, it syncs automatically across all your devices.

iCloud Drive / TradeLogs / journal-backup.json
How to sync
1
Click Export to iCloud above — this downloads a .json file to your device.
2
On Mac: open Finder → iCloud Drive → create a TradeLogs folder → move the file there.
On iPhone: open the Files app → iCloud Drive → TradeLogs → move the file there.
3
iCloud syncs automatically. Open the file from any device to import it back.
4
To restore: click Import from iCloud and select the JSON file from your iCloud Drive.
5
Tip: export after every trading session so your backup is always current.
