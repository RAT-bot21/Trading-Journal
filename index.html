Trade history CSV export
  • 1Open MT5 → press Ctrl+T to open the Toolbox at the bottom
  • 2Click the History tab in the Toolbox
  • 3Right-click anywhere in the history → select Save as Report → choose CSV format
  • 4Alternatively: right-click → Export → save as .csv
  • 5Drop the CSV file below — trades will be mapped and imported automatically

Supports .csv and .htm report formats
MT5 CSV columns recognised: Time, Symbol, Type (buy/sell), Volume, Price, S/L, T/P, Close Time, Close Price, Commission, Swap, Profit, Comment
Duplicate trades (same ticket number) are automatically skipped.

Spot trade history (Buy/Sell orders) is also supported.

MockCapital
MatchTrader / DXtrade / MT5 platform export
MockCapital uses MT5, MatchTrader, or DXtrade as the underlying trading platform. Export your trade history from whichever platform your account runs on, then drop it below.
  • 1MT5 accounts: Open MT5 → Ctrl+T → History tab → right-click → Save as Report → HTM. Use the MT5 tab instead.
  • 2MatchTrader accounts: Log into your MatchTrader dashboard → History → Closed Positions → Export CSV
  • 3DXtrade accounts: Log into DXtrade → Account → Trade History → select date range → Download CSV
  • 4If columns aren't recognised automatically, use the Universal tab to map them manually
Drop MockCapital export file here or click to browse
Supports MT5 HTM, MatchTrader CSV, DXtrade CSV
Columns auto-detected for: MatchTrader (Symbol, Type, Volume, Open Price, Close Price, Open Time, Close Time, Profit, Commission, Swap) and DXtrade (instrument, side, qty, openPrice, closePrice, openTime, closeTime, realizedPnL).
If your file isn't recognised, switch to the Universal tab to map columns manually.

Breakout Prop
Crypto-only prop firm — Kraken-backed
Breakout runs a proprietary trading terminal (the Breakout Terminal). There is no official CSV export button, but you can get your trade data via the steps below.
  • 1Log into your Breakout account at breakoutprop.com
  • 2Go to Trade History or Account → History in the dashboard
  • 3Look for a Download, Export, or CSV button. If available, download and drop below.
  • 4If no export button: right-click the trade table → Select All → Copy → paste into a new Numbers or Excel file → save as CSV → drop below
  • 5Alternatively use the Universal tab and map your columns manually for any format
Drop Breakout Prop CSV here or click to browse
Any CSV with trade data — unmapped columns go to Universal mapper
Breakout columns auto-detected: Symbol/Pair, Side (Buy/Sell/Long/Short), Entry Price, Exit/Close Price, PnL/Profit, Size/Qty, Open Time/Date.
Since Breakout is crypto-only, all imported trades are tagged as Crypto automatically.
