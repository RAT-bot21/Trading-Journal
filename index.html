How to sync
1
Click Export to iCloud — downloads a .json file with everything.
2
Mac: Finder → iCloud Drive → create a TradeLogs folder → move file there.
iPhone: Files app → iCloud Drive → TradeLogs → move file there.
3
To restore on any device: click Import from iCloud and pick the JSON file.
4
Tip: export after every session so your backup stays current.
